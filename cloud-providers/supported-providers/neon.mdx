## Requirements

To set up Neon with DBCode, you will need to use an API key. 

You can obtain your API key by following the instructions provided on the <a href="https://console.neon.tech/app/settings/api-keys" target="_blank">Neon API key setup page</a> in the dashboard. 

## Connecting

Once you have your API key, follow the [connect a cloud provider](/docs/cloud-providers/connect) article.

For more information about Neon, check out their <a href="https://neon.tech" target="_blank">website</a>.
