## Requirements

To set up Turso with DBCode, you will need to use an API token. 

You can obtain your API token by following the instructions provided on the <a href="https://app.turso.tech/api-tokens" target="_blank">API Token</a> page in the dashboard. 

## Connecting

Once you have your API key, follow the [connect a cloud provider](/docs/cloud-providers/connect) article.

For more information about Turso, check out their <a href="https://turso.tech" target="_blank">website</a>.
